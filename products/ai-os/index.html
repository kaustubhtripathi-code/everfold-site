How it works

Goals in. Done tasks out.

You don't operate apps. You state a goal, hand it to an agent, and watch it cross the board.

1

Create an agent

Automation (email · calendar · files), Research (search & synthesis) or Assistant (schedule analysis). One click each.

2

Give it a goal

"Triage my inbox." "Research battery tech." "Find my focus blocks." Plain language — the agent routes itself.

3

Watch it land in Done

Live Kanban, a step-by-step agent message trail, and research reports filed automatically into your knowledge base.

Features

A real OS, not another chat box.

🤖

Three autonomous agents

Automation, Research and Assistant — each with deterministic offline fallbacks, so every task completes even with no API key.

📋

Goal-based Kanban

To Do → In Progress → Done. Manage agents by goals, not terminals and logs.

📚

Research → Knowledge pipeline

Research agents write structured, searchable reports straight into your knowledge base.

💰

Cost meter built in

Per-agent and per-provider token & dollar tracking on every LLM call. No end-of-month surprises.

⌘

Ctrl+K everything

Command palette for navigation, new tasks, new agents and theme switching — no mouse required.

🔌

Dual LLM + full API

Switch Gemini ↔ Claude at runtime. 38 REST endpoints with Swagger docs — script anything the UI does.

Honest by design: external actions (Gmail, Calendar, files, web search) run on safe simulated data until you connect your own credentials. You see the full workflow first; you grant access when you decide.
Get started

Running in two commands.

Python 3.12 + Node 18+. SQLite included. Free Gemini tier supported.

# terminal 1 — backend
python -m uvicorn main:app --port 8000

# terminal 2 — frontend
cd frontend && npm run dev

→ http://localhost:3000
